# Simple Function Example

This example demonstrates generating a simple utility function.

## Prompt

```
Create a TypeScript function called isPrime that checks if a number is prime. It should handle edge cases like negative numbers and return a boolean.
```

## Usage

```bash
quenderin add "Create a TypeScript function called isPrime that checks if a number is prime"
```

Or save to file:

```bash
quenderin add "Create a TypeScript function called isPrime that checks if a number is prime" -o src/gen/isPrime.ts
```
